## Bitwise Approach
In this approach we are essentially going to be using the same login as [[191. Number of 1 Bits]] to count the number of 1 bits for each of the values that i takes on.
``` python
class Solution:
    def countBits(self, n: int) -> List[int]:
        result = [0] * (n + 1)
        for i in range(n + 1):
            num = i
            while num:
                num &= (num - 1)
                result[i] += 1
        return result
```
